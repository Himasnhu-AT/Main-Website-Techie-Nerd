CPP Documentation:

C++ is a general-purpose, compiled, statically typed, free-form programming language. C++ is a middle-level language, meaning that it offers a good balance of abstraction and control. It is possible to write C++ programs that are both efficient and easy to read and maintain. C++ is also a very portable language, meaning that programs written in C++ can be compiled and run on a wide variety of platforms.
